# Physical Computing Project 2025 - IT KMITL
Project Hydroponic farm. For Physical-Computing, IT KMITL. Year 2, semester 1/2025.

## วัตถุประสงค์
### โครงงานนี้คือการพัฒนาฟาร์มไฮโดรโปนิกส์ขนาดเล็กด้วย บอร์ดอาร์ดุยโน(Arduino) ประกอบชิ้นงานและเขียนโปรแกรมเพื่อควบคุมการทำงาน
โครงงานนี้เป็นส่วนหนึ่งของรายวิชา Physical-Computing 06016409
เทอม 1 ปีการศึกษา 2568

## บทคัดย่อ
### โครงงานนี้เป็นโครงงานเพื่อพัฒนาระบบการปลูกพืชโดยใช้น้ำ(ไฮโดรโปนิกส์)ด้วยภาษาซี และประกอบชิ้นงานโดยใข้ไมโครคอนโทรลเลอร์ โครงงานนี้เป็นส่วนหนึ่งของรายวิชา Physical-Computing โดยนักศึกษาชั้นปีที่ 2 เทอม 1 ปีการศึกษา 2568
### ไฮโดรโปนิกส์ เป็นรูปแบบการปลูกพืชโดยการใช้น้ำทดแทนดิน ทำให้ได้รับความนิยมเนื่องจากความสะดวกในการทำการเกษตรในพื้นที่ที่มีความไม่เหมาะสม และสามารถผนวกเข้ากับระบบอื่น ๆ ทั้งเทคโนโลยีในปัจจุบันทำให้เกิดความสะดวกรวดเร็วทั้งในการติดตาม ควบคุม และจัดการ
### ชิ้นงานนี้ประกอบไปด้วยระบบ 3 ส่วน ได้แก่ ระบบน้ำ ระบบไฟ และการแสดงผล โดยระบบน้ำมีการใช้เซนเซอร์วัดระดับน้ำเพื่อการเติมน้ำอัตโนมัติเมื่อน้ำต่ำกว่าระดับที่กำหนด ระบบไฟสำหรับการทำไฟเลี้ยงพืช โดยใช้สวิตช์ในการเปิด-ปิดไฟ การแสดงผลโดยใช้จอLCD เพื่อบอกสถานะน้ำ(ถึงระดับที่กำหนด, กำลังเติมน้ำ) และสถานะไฟ(เปิด, ปิด)
### โครงงานนี้มุ่งเน้นการใช้เทคโนโลยีควบคู่กับการเกษตร เพื่อเพิ่มความสะดวก ทันสมัย เน้นความสะอาดและปลอดภัย เหมาะกับการทำการเกษตรในยุคใหม่ที่ใช้เทคโนโลยีอย่างยั่งยืน

## รายละเอียด
### โครงงานนี้คือการพัฒนาฟาร์มไฮโดรโปนิกส์ขนาดเล็กด้วยภาษาซี (C) โดยใช้
  - บอร์ดอาร์ดุยโน (Arduino)
  - เซนเซอร์วัดระดับน้ำ 
  - ปั๊มน้ำ 
  - ไฟ (LED)
  - สวิตช์
  - จอแสดงผล (LCD) <br>
